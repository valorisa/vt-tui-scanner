# Changelog

All notable changes to VT TUI Scanner will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2024-01-15

### Added
- Initial release
- TUI interface with Textual framework
- File, directory, and URL scanning
- Directory monitoring with rate-limit awareness
- JSON and CSV export functionality
- Docker support
- CI/CD pipeline with GitHub Actions
- Comprehensive test suite
- Cross-platform support (Windows, macOS, Linux)

### Security
- API key management via environment variables
- Secure handling of scan results
- Rate limit enforcement to prevent API abuse

### Documentation
- Complete README with usage examples
- Contributing guidelines
- Code of conduct

## [Unreleased]

### Planned
- [ ] GUI mode option (optional PyQt/Tkinter)
- [ ] Scheduled scans with cron-like syntax
- [ ] Multi-threaded scanning for large directories
- [ ] Integration with other threat intelligence platforms
- [ ] Plugin system for custom scanners
